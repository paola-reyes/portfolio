Exploring Urban Data

Finding ways to visualize spatio-temportal data in a 3D environment.

Overview

This research project aimed to find the best tools to support the Urban Toolkit's (UTK) information design. Embedding plotted graphs over a 3D map enhances the UTK's use cases for urban data visualization.

Our Research

The Urban Toolkit, a 3D visualization framework for urban data visualization.

This project expands upon the Urban Toolkit's (UTK) capabilties in portraying data related to urban enviromnents. The UTK allows users to input data files to be generated as thematic data onto a 3D map of the city's landscape. The goal of our findings was to allow for comparing data from two different sources onto a 2D graph, like a scatterplot, to be embedded over the 3D map.

Merging datasets for 2D visualization using Vega-Lite.

Our research focused on creating new possibilities for urban data researchers. The proposed functionality allowed for more information (e.g., comparing city statistic categories) to be present in a UTK map that previously supported one dataset at a time. Uploading this information into the UTK will connect thematic community statistics to a city's geographic boundaries. Merged datasets can be used to generate interactive 2D Vega-Lite graphs that supplement the map data.

Working in affiliation with the Early Research Scholars Program.

This research was completed under the guidance of UIC's Early Reasearch Scholars Program (ERSP). The UTK framework project is an effort of the UIC Electronic Visualization Laboratory. We presented our research during a lightning-talk at the ERSP National Conference .
